# ✅ 프로젝트 목표
### ✔ 현대 사회의 독서량 부족 문제 해결과 지식 함양을 목표로 한다.
### ✔ 주 소비층인 20, 30대를 겨냥하여 SNS 기능을 도입함으로써 젊은 사람들의 흥미를 유발 시&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;킨다.
### ✔ 다양한 API를 활용하여 큐레이팅 서비스를 제공함으로 사용자 경험 개선과 안정적인 수익 &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;모델을 보장한다.

# 👥 프로젝트 임무 분담
* #### 팀장
  - 메인, 제품, 카테고리 페이지 제작
  - API(유튜브, 네이버)를 활용한 큐레이팅 서비스

* #### 개발 팀장
  - 장바구니
  - 주문페이지 및 주문내역 페이지
  - 결제 API
  - 게시판 댓글 기능
 
* #### 부팀장
  - 자유 게시판
  - Q&A 게시판

* #### 팀원 1
  - 관리자 페이지
  - open api (AI 챗봇)
  - 공지사항
  - 책 검색 기능

* #### 팀원 2
  - 로그인(소셜) 및 회원가입 기능
  - 마이페이지
 
* #### 팀원 3
  - 책 리뷰 기능
  - SNS 기능
